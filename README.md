# pcp-ui

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

## 组件
``` bash
# 安装 Element-ui
npm install element-ui

# 安装 axios
npm install axios

# 安装 QS
npm install qs


```
